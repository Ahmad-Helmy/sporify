Reset your password

An elephant never forgets. For everyone else there's password reset.

  1. Go to password reset.
  2. Enter your email address and click SEND.
  3. Go to your email inbox and find an email from us with the subject “Reset your password”.
    Tip: If you don’t see it, check your junk/spam folders or any other filtered folders (e.g. If you use Gmail, it might be in the Social or Promotions tabs).
  4. Click the Reset password link in the email. A new web page will open.
  5. Type your new password in both fields and click SEND.
  6. Go back to the app and log in with the email address you used to reset your password and the new password.

Didn’t work?

If you see the error message "The email address you entered is not connected to a Mayestro account", try the above steps again with any other email addresses you have.

Tip: Check your email inboxes for emails from Mayestro. If you have any, it’s likely there’s an account associated with that email address.

If you don’t have access to an email address associated with your Mayestro account, you’re unable to reset the password, so you need to create a new account.

If the password reset submitted successfully, but you can’t find the email:

  • Check your spam/junk folders.
  • Check any other filtered folders (e.g. if you use Gmail, it might be in the Social or Promotions tabs).

If you don’t have access to an email address associated with your Mayestro account, you’re unable to reset the password, so you need to create a new account.

If you receive the message “password reset link is not valid, or already used” try submitting for a new reset link and, without clicking the link, follow these steps:

  1. Go to your password reset email.
  2. Right-click the Reset password link and select Copy link address.
    Note: If you accidentally click the link before you do this, you’ll have to send for a new one.
  3. Go to your web browser and open a private/incognito window.
  4. Paste the link into the address bar.
  5. Press Enter and continue.
