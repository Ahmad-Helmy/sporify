Can't log in

Username, password, or email address incorrect?

Use our password reset tool.

Here’s some help if you still can’t get in:

No Mayestro account linked to email address?

If you see this error message, try again with any other email addresses you have.

Tip: Check your inboxes for emails from Mayestro. If you have any, there’s likely an account associated with that email address.

If you can’t find an email address connected to a Mayestro account, your account might be connected to Facebook. On the log in page, try LOG IN WITH FACEBOOK.

Don’t have access to your account’s email address?

In this case, you’re unable to get the link to reset your password. But you do have a few options:

  • Regain access to the email address if possible.
  • Contact us and we can update the email address for you.
  • Create a new account and start fresh.

Password reset link doesn’t work?

If you receive the message “password reset link is not valid, or already used” try sending a new password reset link and, without clicking the link in your email, follow these steps:

  1. Go to your password reset email.
  2. Right-click (desktop) or press and hold (mobile) the Reset password link and select Copy link.
    Note: If you accidentally click the link before you do this, you’ll have to send for a new one.
  3. Go to your web browser and open a private/incognito window.
  4. Paste the link into the address bar.
  5. Press Enter and continue.

Not received password reset email?

If the password reset submitted successfully, but you can’t find the email:

  • Check your spam/junk folders.
  • Check any other filtered folders (e.g. if you use Gmail, it might be in the Social or Promotions tabs).

Can’t log in with your phone number?

Here are some tips if you're having trouble logging in with your phone number:

  • If you see the message ‘Try again’, it’s likely you’ve entered the number incorrectly. Check the number and try again.
  • If the verification code isn’t working, check the number and try again, or re-enter your phone number to generate a new code.
  • If you signed up with an email address or Facebook, you can log in by using the CONTINUE WITH EMAIL or CONTINUE WITH FACEBOOK button.

If you try to sign up/log in with a phone number, but don’t receive a verification code:

  • Wait a few minutes. You should receive a code almost immediately, but if your reception is limited it might take a little longer.
  • Generate a new code by re-entering your phone number. Be sure to check you enter the number correctly.

New phone number?

If you signed up with a phone number but have since changed it, contact us and we can update the number that’s associated with your account.
